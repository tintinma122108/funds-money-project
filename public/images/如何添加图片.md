# 如何添加图片文件

## 当前状态
图片文件尚未保存到目录中。请按照以下步骤操作：

## 步骤 1：准备图片文件
您需要准备三张 PNG 格式的图片：
1. **crumpled-ball.png** - 纸团图片
2. **binder-clip.png** - 夹子图片  
3. **paper-clip.png** - 回形针图片

## 步骤 2：保存图片到目录
将三张图片文件直接保存（或复制）到以下目录：

```
haze-soup/public/images/
```

## 步骤 3：验证文件
保存后，目录应该包含：
```
public/images/
├── crumpled-ball.png
├── binder-clip.png
├── paper-clip.png
└── README.md
```

## 步骤 4：重启开发服务器
保存图片后，请重启 Next.js 开发服务器：
```bash
# 停止当前服务器（Ctrl+C）
# 然后重新启动
npm run dev
```

## 图片要求
- 格式：PNG（支持透明背景）
- 文件名：必须完全匹配（区分大小写）
- 尺寸：代码会自动处理，但建议：
  - crumpled-ball.png: 宽度约 340px
  - binder-clip.png: 宽度约 96px
  - paper-clip.png: 82px × 146px
