 # 基金估值助手 APP - 设计风格指南 (Design Style Guide)

> 本文档定义了基金估值助手应用的整体设计风格、视觉语言和组件样式规范。设计团队和开发团队应遵循本指南确保设计一致性。

---

## 1. 设计风格主题

### 1.1 设计理念
**主题**：复古工业风 Paper 设计风格

**核心理念**：
- 模拟真实纸张的质感和纹理
- 结合工业风格的装饰元素
- 营造专业、可信赖的金融工具感
- 通过细节传达品质和精致感

### 1.2 视觉语言
- **质感**：纸张纹理、折痕、噪点
- **装饰**：回形针、夹子、撕纸边缘
- **色彩**：米白色背景、深色文字、红绿收益标识
- **字体**：科技感数字字体 + 优雅标题字体
- **动画**：流畅的数字滚动、纸张翻转效果

## 2. 设计元素

### 2.1 纸张纹理系统

#### 背景纸张纹理
```css
/* 纸张纹理叠加层 */
background-image: url("data:image/svg+xml,...");
opacity: 0.03;
background-repeat: repeat;
```

**应用场景**：
- 页面主背景
- 卡片背景（可选）

**视觉效果**：
- 微妙的噪点纹理
- 模拟真实纸张的质感
- 不干扰内容阅读

#### 纸张折痕效果
```css
/* 微妙的纸张折痕 */
repeating-linear-gradient(
  90deg,
  transparent,
  transparent 79px,
  rgba(0,0,0,0.03) 79px,
  rgba(0,0,0,0.03) 80px
),
repeating-linear-gradient(
  0deg,
  transparent,
  transparent 79px,
  rgba(0,0,0,0.02) 79px,
  rgba(0,0,0,0.02) 80px
);
opacity: 0.02;
```

**应用场景**：
- 页面背景叠加层
- 增强纸张真实感

#### 卡片纸张背景
- **颜色**：纯白色 (#ffffff) 或米白色 (#fbfaf8)
- **纹理**：可选的轻微纸张纹理
- **阴影**：模拟纸张浮起的阴影效果

### 2.2 装饰元素

#### 回形针装饰 (Paper Clip)
**位置**：总持仓卡片顶部左侧
**尺寸**：响应式，`clamp(120px, 10vw, 164px)`
**行为**：
- 默认状态：轻微旋转（-16deg）
- 展开状态：旋转并移动（-60deg）
- 动画：0.7s 弹性缓动

**设计要点**：
- 增强纸张文件的真实感
- 提供视觉趣味性
- 不影响内容阅读

#### 夹子装饰 (Binder Clip)
**位置**：总持仓卡片右侧
**尺寸**：210px 宽度
**行为**：
- 默认状态：可见
- 展开状态：隐藏（透明度动画）
- 动画：0.5s 平滑过渡

**设计要点**：
- 模拟文件整理的真实场景
- 增强工业风格感

#### 撕纸边缘效果 (Torn Edge)
**位置**：基金卡片底部
**实现方式**：使用 `clip-path` 创建锯齿边缘

```css
clip-path: polygon(
  0 0,
  100% 0,
  100% calc(100% - 16px),
  /* 锯齿边缘点 */
  calc(100% - 4%) calc(100% - 16px),
  calc(100% - 8%) calc(100% - 8px),
  /* ... 更多锯齿点 ... */
  0 calc(100% - 16px)
);
```

**设计要点**：
- 模拟从纸张上撕下的效果
- 增强手工感和真实感
- 每个卡片都有独特的边缘（可选）

## 3. 布局规范

### 3.1 页面布局

#### 容器结构
```css
/* 主容器 */
max-width: 1280px;
margin: 0 auto;
padding: 32px;
```

**响应式处理**：
- 使用 `clamp()` 函数实现响应式内边距
- 移动端自动适配

#### 内容层级
1. **Slogan 区域**：页面顶部，大标题
2. **总持仓卡片**：固定位置，页面顶部
3. **基金列表**：主要内容区域
4. **添加按钮**：固定在列表标题栏

### 3.2 卡片布局

#### 总持仓卡片 (Portfolio Header)
**布局结构**：
```

│  总资产 (CNY)               │
│  [大数字]                    │
│  ─────────────────────      │
│  累计收益  │  当日盈亏      │
│  [数字]    │  [数字]        │
└─────────────────────────────┘
```

**尺寸规范**：
- 内边距：24px
- 最小高度：280px
- 右侧内边距：64px（为夹子装饰留空间）

**交互状态**：
- 默认：显示遮罩数据（****）
- 展开：显示真实数据
- 3D翻转动画：0.7s

#### 基金卡片 (Fund Card)
**布局结构**：
```
┌─────────────────────────────┐

│  估算涨跌                   │

│  ─────────────────────      │
│  持仓成本  │  持有收益      │
└─────────────────────────────┘
```

**尺寸规范**：
- 内边距：32px
- 底部内边距：40px（为撕纸边缘留空间）
- 卡片间距：24px（垂直方向）

**信息层级**：
1. **主要信息**：基金名称、估算涨跌（大字号）
2. **次要信息**：持有金额、持有收益（中等字号）
3. **辅助信息**：标签文字（小字号）

## 4. 组件样式规范

### 4.1 按钮组件

#### 添加按钮 (Add Button)
**样式特征**：
- 胶囊形状（border-radius: 9999px）
- 红色渐变背景 (#ea553a → #d64126)
- 多层阴影效果
- 纸张纹理叠加

**交互状态**：
- **默认**：正常显示
- **悬停**：轻微上浮
- **按下**：缩放和阴影变化

**尺寸**：
- 高度：40px
- 宽度：96px
- 内边距：12px 40px

#### 删除按钮 (Delete Button)
**样式特征**：
- 透明背景
- 图标颜色：灰色 (#d1d5db)
- 圆形点击区域

**交互状态**：
- **默认**：灰色图标
- **悬停**：红色图标 + 浅红背景 (#fef2f2)

### 4.2 输入框组件

#### 模态框输入框
**样式特征**：
- 透明背景
- 底部边框（1px solid #d6d3d1）
- 居中对齐
- 等宽字体（Share Tech Mono）

**交互状态**：
- **默认**：灰色边框
- **聚焦**：深色边框 (#1c1917)
- **占位符**：灰色文字

**标签样式**：
- 小字号（10px）
- 大写字母
- 字间距加宽（0.2em）
- 灰色文字 (#a8a29e)

- **关闭（取消）**：揉皱动画（0.8s）

## 5. 动画效果规范

### 5.1 数字滚动动画 (RollingNumber)

#### 动画特点
- **类型**：垂直滚动翻页效果
- **效果**：类似老虎机或里程表
- **实现**：每个数字位（0-9）独立滚动

#### 应用场景
- 总资产数字更新
- 累计收益更新
- 当日盈亏更新
- 基金卡片中的所有数字更新

#### 动画参数
- **时长**：0.8秒
- **缓动**：cubic-bezier(0.4, 0.0, 0.2, 1)
- **触发**：数值变化时自动触发

#### 视觉表现
- 数字从旧值滚动到新值
- 支持小数位、正负号、百分比
- 配合 Share Tech Mono 字体，增强科技感

### 5.2 卡片交互动画

#### 悬停效果
- **位移**：向上移动 4px
- **时长**：0.3秒
- **缓动**：ease-out
- **阴影**：轻微增强（可选）

#### 添加动画
- **类型**：从下方淡入
- **位移**：从 y: 20px 到 y: 0
- **透明度**：从 0 到 1
- **时长**：0.3秒
- **延迟**：根据索引递增（stagger）

#### 删除动画
- **类型**：淡出
- **透明度**：从 1 到 0
- **时长**：0.3秒
- **可选**：轻微缩放

### 5.3 总持仓卡片动画

#### 3D翻转效果
- **触发**：点击卡片
- **效果**：绕 Y 轴旋转 -140deg
- **时长**：0.7秒
- **缓动**：cubic-bezier(0.23, 1, 0.32, 1)

**实现细节**：
- 使用 `transform-style: preserve-3d`
- 正面：遮罩数据
- 背面：空白纸张
- 翻转时显示真实数据

#### 装饰元素动画
- **回形针**：旋转和移动（0.7s 弹性）
- **夹子**：淡出和移动（0.5s 平滑）

### 5.4 模态框动画

#### 打开动画
- **类型**：缩放 + 上移
- **初始**：scale: 0.95, y: 20, opacity: 0
- **结束**：scale: 1, y: 0, opacity: 1
- **时长**：0.4秒
- **缓动**：弹性（cubic-bezier(0.34, 1.56, 0.64, 1)）

#### 关闭动画（确认）
- **类型**：向下翻转
- **效果**：y: 150, rotateX: -90, scale: 0.9
- **时长**：0.5秒
- **缓动**：回弹（cubic-bezier(0.68, -0.55, 0.265, 1.55)）

#### 关闭动画（取消）
- **类型**：揉皱效果
- **效果**：纸张揉皱图片动画
- **时长**：0.8秒
- **缓动**：ease-in-out

## 6. 信息层级设计

### 6.1 视觉层级

#### 第一层级（最重要）
- **总资产数字**：48px，深色
- **估算涨跌幅**：36px，红/绿色
- **基金名称**：18px，深色

#### 第二层级（重要）
- **累计收益/当日盈亏**：24px，红/绿色
- **持有金额/持有份额**：18px，深色
- **持有收益**：18px，红/绿色

#### 第三层级（辅助）
- **标签文字**：12px，灰色
- **基金代码**：14px，灰色
- **收益率**：14px，红/绿色

### 6.2 信息分组

#### 总持仓卡片分组
1. **总资产**：独立区块，最大字号
2. **收益信息**：网格布局，两列显示

#### 基金卡片分组
1. **标题区**：基金名称 + 代码 + 删除按钮
2. **核心数据区**：估算涨跌（突出显示）
3. **详细信息区**：网格布局，多行显示

### 6.3 信息亲密度原则

#### 相关元素靠近
- 标签和数值：紧密排列（8px 间距）
- 相关数据：使用网格布局，统一对齐
- 分割线：用于区分不同信息组

#### 无关元素分离
- 卡片之间：24px 间距
- 区块之间：32px 间距
- 使用分割线或边框区分

## 7. 响应式设计

### 7.1 断点策略
- **移动端**：< 640px
- **平板**：640px - 1024px
- **桌面**：> 1024px

### 7.2 响应式元素

#### 字体大小
```css
font-size: clamp(36px, 8vw, 72px);  /* Slogan */
font-size: clamp(20px, 4vw, 32px);   /* 标题 */
```

#### 间距
```css
padding: clamp(16px, 4vw, 32px);     /* 页面内边距 */
margin: clamp(16px, 4vw, 64px);      /* 区块间距 */
```

#### 装饰元素
- 回形针：`clamp(120px, 10vw, 164px)`
- 夹子：响应式定位

### 7.3 布局适配
- **移动端**：单列布局，减少装饰元素
- **平板**：保持桌面布局，调整间距
- **桌面**：完整布局，所有装饰元素

## 8. 可访问性 (Accessibility)

### 8.1 颜色对比度
- **主要文字**：与背景对比度 ≥ 4.5:1
- **辅助文字**：与背景对比度 ≥ 3:1
- **交互元素**：清晰的视觉反馈

### 8.2 交互反馈
- **按钮**：明确的悬停和按下状态
- **输入框**：聚焦状态清晰可见
- **删除操作**：确认对话框防止误操作

### 8.3 语义化
- 使用语义化 HTML 标签
- 提供适当的 ARIA 标签
- 键盘导航支持
