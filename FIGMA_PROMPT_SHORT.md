# Figma AI Prompt（可直接使用）

```
设计一个基金估值助手移动端界面，参考Robinhood的大空间简约风格。

## 设计风格
- 深色主题（黑色背景#000000）
- 大空间设计，充足留白
- 简约风格，去除不必要装饰
- 透明灰色卡片（zinc-500/10，毛玻璃效果backdrop-blur）
- 大圆角（16px），极细边框（white/10）
- 红色涨（#fb7185），绿色跌（#34d399）

## 界面结构

### 1. 总持仓模块（固定顶部）
居中垂直布局，大间距（24px）：
- 总持仓金额：40px大号白色数字，带动画
- 持仓总收益：20px，红色/绿色，带动画
- 估算今日总收益：20px，红色/绿色，带动画  
- 估算今日涨跌：24px，红色/绿色，带图标，带动画
底部渐变分割线

### 2. 添加按钮（右上角固定）
48×48px圆形，绿色背景，白色加号图标，带阴影

### 3. 基金列表（可滚动）
每个基金独立卡片slot：
- 卡片：透明灰色背景（zinc-500/10），毛玻璃效果，16px圆角，20px内边距，底部16px间距
- 内容从上到下：
  1. 基金名称（15px白）+ 代码（12px灰），右侧删除按钮
  2. 持有金额：20px大号白色，带动画
  3. 持有收益区域（2列网格，20px间距）：
     - 持有收益：15px，红/绿，带动画
     - 持有收益率：15px，红/绿，带动画
  4. 今日估算区域（2列网格，顶部分割线）：
     - 估算今日收益：15px，红/绿，带动画
     - 估算今日涨跌：15px，红/绿，带涨跌图标，带动画

信息亲密度：相关元素紧密（20px），不同组间距大（20px）

### 4. 添加基金模态框
深色遮罩+居中卡片，深灰背景，表单：基金代码、持仓金额、持仓成本价

## 交互细节
- 每2秒数据刷新，数字变化时平滑动画过渡（0.65秒）
- 卡片悬停背景加深
- 按钮点击轻微缩放反馈

## 字体
- 超大：40px（总金额）
- 大：24px（今日涨跌）
- 中：20px（持有金额）
- 标准：15px（收益数值）
- 小：12px（标签）
数字使用等宽字体（tabular-nums）

## 间距
- 超大：24px（主要区块）
- 大：20px（卡片间距、组内元素）
- 中：16px（相关元素）
- 小：12px（标签数值）

请设计高保真界面，包含总持仓模块、基金slot详细设计、模态框、空状态和加载状态。
```
